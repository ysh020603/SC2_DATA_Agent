# Terran Unit Keys

Common Terran producer keys:

- `CommandCenter`: town hall and SCV producer.
- `Barracks`: infantry production building.
- `Factory`: vehicle production building.
- `Starport`: air unit production building.
- `EngineeringBay`: infantry/structure upgrade building.
- `GhostAcademy`: Ghost and related tech building.
- `Armory`: vehicle/ship upgrade building.
- `FusionCore`: Battlecruiser tech building.
